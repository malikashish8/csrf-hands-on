# CSRF Hand-on

Learn CSRF practically through various use cases. Use cases are listed in [web](/web/) folder.

## Run

- Open 2 bash tabs
- in 1 tab run
  
  ```go run domain1.go```
- in 2nd run
  
  ```go run domain2.go```
- open <https://localhost:8001> in a private/incognito browser window

## Prerequisites

- Go 1.11
